Now

Building acaua — an Apache-2.0, PyTorch-native computer vision library. One API (predict / train / export) across detection, segmentation, OBB, OCR, VLMs, and zero-shot.

The goal: a pythonic alternative to mmdet and PaddleDetection — fewer abstractions, less YAML, friendlier extras like acaua[ocr], acaua[vlm], acaua[zeroshot], acaua[export].
